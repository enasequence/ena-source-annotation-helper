ENA Source Attribute Helper

ENA Source Attribute Helper (SAH) tool is backed by the RESTful API Architecture and it is designed to use the data available in the NCBI BioCollections. NCBI BioCollections is a curated database of metadata for natural history collections, associated with records in INSDC, that includes the institution and collection codes.

SAH tool help users accurately report biological source-related sequence and sample attributes. This tool currently focuses on the attributes in which specimens, cultures or other materials are identified, from which the sequence data were derived, and uses curated data to obtain the unique codes for the institutions and collections holding the vouchers.

The tool's main functions include the querying of the metadata for the institutions and collections based on the user input, validation of institution and collection codes in the attribute strings provided by the user, and the construction of the attribute string based on the user-provided information.

APIs can be accessed directly using SAH API URL

Note: The SAH tool does not include the search or validation of the voucher specimen codes.
